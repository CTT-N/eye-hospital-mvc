Design — Implementation Order

Recommended sequence to integrate all 25 pages safely

Phase 1 — Foundation (do first, everything depends on this)

1. Move fe/css/, fe/js/, fe/images/ → webapp/static/
2. Add JSTL jar to webapp/WEB-INF/lib/ (jakarta.servlet.jsp.jstl-3.0.jar)
3. Convert login.html → login.jsp (fix form action, test auth end-to-end)
4. Convert register.html → register.jsp

Phase 2 — Patient role (5 pages)

5. patient-dashboard.jsp + PatientDashboardController (show name, upcoming appts count)
6. patient-appointments.jsp + PatientAppointmentController (list + cancel)
7. patient-history.jsp + PatientHistoryController (past records)
8. patient-invoices.jsp + PatientInvoiceController (billing list)
9. patient-profile.jsp + new PatientProfileController

Phase 3 — Doctor role (5 pages)

10. doctor-dashboard.jsp + DoctorDashboardController
11. doctor-patient-list.jsp + PatientListController
12. doctor-patient-record.jsp + MedicalRecordController
13. doctor-schedule.jsp + DoctorScheduleController
14. doctor-appointment-detail.jsp + ExaminationController
15. doctor-profile.jsp + new DoctorProfileController

Phase 4 — Manager role (5 pages)

16. manager-dashboard.jsp + ManagerDashboardController
17. manager-departments.jsp + DepartmentController
18. manager-rooms.jsp + RoomController
19. manager-schedules.jsp + ScheduleController
20. manager-hospital-info.jsp + HospitalController
21. manager-reports.jsp + ReportController

Phase 5 — Admin role + Public pages

22. admin-dashboard.jsp + AdminDashboardController
23. admin-users.jsp + ManageUserController
24. admin-eye-diseases.jsp + ManageEyeDiseaseInfoController
25. find-doctor.jsp + new FindDoctorController (public)
26. book-appointment.jsp + PatientAppointmentController
27. home.jsp (mostly static, minimal data)
